28 Ιανουαρίου 1996

 

Ο Θεός σού στέλνει την παρατεταμένη αυτή περίοδο δοκιμασιών για να βεβαιωθεί πως η δύναμη δεν θα σε καταστρέψει. Η δύναμη που θα αποκτήσεις είναι μεγάλη, και πρέπει να εκπαιδευτείς αυστηρά για να μάθεις να μπορείς να τη χειρίζεσαι. Χρειάζεσαι πολλές ασκήσεις για να ισχυροποιηθείς τόσο, που να μην κινδυνεύεις να αυτοκαταστραφείς από τη δύναμη.

Το να μάθεις να είσαι ολιγαρκής ως προς τα χρήματα σημαίνει πως θα μπορείς να τους αποδώσεις την πραγματική τους αξία όταν τα έχεις, και να μην είσαι εξαρτημένη απ' αυτά. Το ίδιο γίνεται αντίστοιχα και με πνευματικά θέματα.

Η ζωή σου είναι διαφορετική από των άλλων, αυτό πρέπει να το δεχτείς. Μη φοβάσαι, απλώς καλείσαι να ακολουθήσεις τα λόγια που ακούς, και να μην παρασύρεσαι από τα πλάνα ζωής που ισχύουν για τους άλλους ανθρώπους.

Οι πειρασμοί σε βομβαρδίζουν, αλλά πρέπει να μπορείς να τους αναγνωρίζεις για να μη χάνεις χρόνο. Έχεις πολύ χρόνο μπροστά σου, το έργο σου θα είναι ιδρυτικό, χωρίς όμως να πρόκειται για πρακτική συσπείρωση. Ρίχνεις τους σπόρους της αφύπνισης και εκείνοι φυτρώνουν.

Το …. σου έργο είναι εξίσου σημαντικό και απαραίτητο, όπως θα διαπιστώσεις.

Η Ελλάδα είναι η νέα κοιτίδα του πολιτισμού του φωτός. Σε πλησιάζουν οι άνθρωποι που θα συμπράξουν στην αναγέννηση. Το βλέπεις και να το περιμένεις.

Με γνωρίζεις από άλλες ζωές. Θα βρίσκομαι κοντά σου για τη νέα φάση και θα σε καθοδηγώ. Μίλα σε όσους στο ζητάνε, μέσα απ' τα λόγια σου θα μιλάει ο νέος κόσμος που θα ανατείλει σε λίγο. Όσες επιθέσεις κι αν δεχτείς, μην υποχωρήσεις. Θα περιβληθείς μανδύα φωτός που θα καλέσει γύρω σου τους «νέους» ανθρώπους.

Ψάξε στο μακρινό παρελθόν για να με βρεις. Ας είναι αυτή η μέρα ενός νέου δρόμου. Ο Θεός σου αναθέτει ένα δύσκολο έργο, ο Θεός σε ξεχωρίζει και σε ευλογεί για την ταπεινή και καλή σου καρδιά.

Θα είμαι πάλι εδώ για να σε κατευθύνω.
