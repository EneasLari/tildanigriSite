1.4 Η τέχνη της απόκρυψης


«Τα τετράστιχά μου», λέει ο Νοστράδαμος, «δεν είναι γραμμένα αποκλειστικά στα γαλλικά, υπάρχουν αρκετές επιρροές και από τα Λατινικά.
Πολύ συχνά, στη γραφή μου χρησιμοποίησα τον αναγραμματισμό. Ένας λόγος είναι διότι δεν ήθελα να προκαλέσω τους πολιτικούς της εποχής μου. Στην εποχή μου, οι ευγενείς είχαν μεγάλη δύναμη. Θα μπορούσαν να με συλλάβουν και τότε θα τελείωναν όλα. Έτσι είμαι πρόθυμος να κάνω ό,τι μπορώ για να συγκαλύψω αυτά που γράφω, αρκεί να τα γράφω.

Ένας άλλος λόγος που σε κάποια άλλα τετράστιχα χρησιμοποιώ αναγραμματισμούς είναι διότι το βασικό θέμα είναι πολύ ευαίσθητο για την εποχή για την οποία γράφεται. Δεν θα ήταν καλό για το γενικότερο κοινό να γνωρίζει αυτό για το οποίο γράφω, γιατί θα μπορούσε να προκαλέσει αναταραχή ή και πανικό. Όσοι γνωρίζουν αυτά τα θέματα θα μπορέσουν και να τα καταλάβουν΄ διότι αυτοί που γνωρίζουν είναι συνήθως οι μόνοι σε θέση να κάνουν κάτι για αυτό.
Οι σκεπτικιστές με ανέκριναν συχνά τόσο για το έργο μου με τους ασθενείς όσο και για τις προφητείες. Οι σκεπτικιστές είναι επίμονοι σαν τα αγριόχορτα. Πρέπει να καταλάβετε ότι είχα να αντιμετωπίσω περισσότερη άγνοια από όση έχετε ποτέ μπορέσει να φανταστείτε. Ακόμα, έχω δεχθεί και κάποιες απειλές από την εκκλησία.
Προσπάθησαν να με κάνουν να μη δημοσιεύσω κάποιες προβλέψεις ή να με παγιδέψουν με κάποιες από τις προβλέψεις μου, προσπαθώντας να τις κάνουν να φαίνονται αιρετικές. Προσπάθησαν να με δωροδοκήσουν και να με εκβιάσουν. Προσπάθησαν ακόμα να με κάνουν να αλλάξω κάποιες προβλέψεις προσαρμόζοντάς τες στις ανάγκες τους. Η εκκλησία δεν είναι ένας θρησκευτικός οργανισμός. Είναι ένας από τους μεγαλύτερους πολιτικούς οργανισμούς στη γη. Η εκκλησία δεν δίνει δεκάρα για τα θρησκευτικά θέματα, τα χρησιμοποιεί για να ενισχύσει την πολιτική χειραγώγηση, ενεργώντας πάντα για το συμφέρον της.

Οι ιερείς, αν και προσπάθησαν να αλλάξουν τα τετράστιχά μου, δεν το κατόρθωσαν. Είναι έτσι γραμμένα που όσοι τα διαβάσουν δε θα μπορούν να βγάλουν νόημα. Συνεπώς, δεν ήξεραν τι θa έπρεπε να αλλάξουν ώστε να εξυπηρετεί τους σκοπούς τους.
Στην πορεία των ετών βέβαια, τα τετράστιχα έχουν υποστεί αλλαγές και διαστρεβλώσεις από εκδότες, επιμελητές κειμένων και μεταφραστές εξαιτίας μάλλον άγνοιας παρά εσκεμμένων λαθών. Τα τετράστιχα έχουν κατ' εξακολούθηση ερμηνευτεί εσφαλμένα, επειδή οι ερμηνευτές νόμιζαν ότι αναφερόμουν σε κάποιο γεγονός που θα συνέβαινε σε μία συγκεκριμένη πόλη ενώ, στην πραγματικότητα, χρησιμοποιούσα συμβολικά τα ονόματα πόλεων για να υποδείξω τις αντίστοιχες χώρες. Παράλλαζα μ' αυτόν και με άλλους τρόπους, τα τετράστιχά μου, ώστε η Ιερά Εξέταση να νομίζει ότι κάνω απλώς αναφορές στην αρχαία Ιστορία.

Θέλω επίσης να παρατηρήσω, με κάποια δόση χιούμορ, ότι ο,τιδήποτε είναι αυτό που θα προβληματίσει την ανθρωπότητα, θα τραβήξει για περισσότερο καιρό την προσοχή της. Δύο παραδείγματα είναι η Ατλαντίδα και εγώ. Τα τετράστιχά μου δεν θα είχαν επιβιώσει ώς τις μέρες σας αν ήταν γραμμένα σε απλή γλώσσα. Όσο για την ερμηνεία τους, θα δείτε ότι από τη στιγμή που αντιληφθείτε την κατεύθυνση που παίρνει ένα τετράστιχο, τις περισσότερες φορές είναι θέμα απλής εφαρμογής στη λογική πρόοδο των γεγονότων. Έτσι, θα πρέπει να είναι εύκολο για ένα λογικό μυαλό να τα κατανοήσει.»

Καθισμένος μόνος τη νύχτα σε μυστική μελέτη,
αυτό είναι τοποθετημένο πάνω στον χάλκινο τρίποδα΄
μια αδύναμη φλόγα βγαίνει από το κενό,
και κάνει επιτυχημένο εκείνο που δεν είναι μάταιο να γίνει πιστευτό. (I-1)


Εδώ δίνω μία περιγραφή του τι έκανα στο εργαστήριό μου. Ξεκίνησα μ' αυτό το τετράστιχο για να εξηγήσω από πού λάβαινα αυτά που έγραφα.

Το ραβδί στο χέρι είναι τοποθετημένο στο κέντρο των ποδιών του τρίποδα,
με νερό ραντίζει την άκρη του μανδύα και τα πόδια του,
μια φωνή, φόβος΄ τρέμει μέσα στα ρούχα του,
θεϊκό μεγαλείο, ο θεός στέκεται δίπλα του. (I-2)


Το πρώτο τετράστιχο εξηγεί τα όργανα που χρησιμοποιώ και το δεύτερο τη διαδικασία με την οποία ανοίγω τον εαυτό του στα ανώτερα πεδία. Ο φόβος δεν σημαίνει ότι φοβάμαι, αλλά ότι σέβομαι. Αυτά που βλέπω μου προκαλούν δέος γιατί δεν κατανοώ όλα όσα συμβαίνουν. Και γνωρίζω ότι οφείλω να εργαστώ με προσοχή για να είμαι σίγουρος ότι δεν θα κάνω κάποιο λάθος.
