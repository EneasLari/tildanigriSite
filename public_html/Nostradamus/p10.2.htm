10.2 Η Γυναίκα και το Ανδρόγυνο

Οι ουρανοί θρηνούν πολύ για τη γέννηση του Ανδρόγυνου,
κοντά στον ουρανό ανθρώπινο αίμα χυμένο.
Είναι πολύ αργά για το μεγάλο έθνος να αναγεννηθεί εξ αιτίας του θανάτου
γρήγορα, αν και πολύ αργά έρχεται η αναμενόμενη βοήθεια. (ΙΙ-45)

Η χρησιμοποίηση της έκφρασης «γέννηση του Ανδρόγυνου» αφορά κάποιες κοινωνικές αλλαγές που θα προκληθούν από την ίδρυση των διαστημικών αποικιών. Ο μόνος τρόπος που θα μπορέσουν αυτές οι αποικίες να λειτουργήσουν είναι με το να εργάζονται αρμονικά άνδρες και γυναίκες μαζί και να ξεπεράσουν τη στενομυαλιά τους και την σωβινιστική συμπεριφορά για τους ρόλους του κάθε φύλου. Η γέννηση του ανδρόγυνου συμβολίζει το ότι τόσο οι άνδρες όσο και οι γυναίκες θα μπορούν να κάνουν ό,τι κρίνουν καλύτερο για τους εαυτούς τους χωρίς να ανησυχούν για το αν θα είναι αρεστό στους άλλους. Αυτή η ριζική κοινωνική αλλαγή θα επηρεάσει τους ανθρώπους που έχουν απομείνει στη γη.

Αναταραχή θα προκληθεί σε κάποια από τα έθνη, όπως για παράδειγμα στη Μέση Ανατολή που θα πρέπει να επεξεργαστούν ιδιαίτερα τα πιστεύω τους πριν μπορέσουν να δεχτούν αυτές τις νέες ιδέες. Επίσης, την εποχή αυτή, θα υπάρξουν πολλά διαστημικά ατυχήματα καθώς οι άνθρωποι θα προσαρμόζονται στον νέο τρόπο ζωής΄ πολλοί πιλότοι θα σκοτωθούν - «κοντά στον ουρανό ανθρώπινο αίμα χυμένο».

Η εξασθενημένη φωνή μιας γυναίκας ακούγεται κάτω από το ιερό έδαφος.
Η ανθρώπινη φλόγα λάμπει για τη θεϊκή φωνή.
Θα κάνει τη γη να κηλιδωθεί από το αίμα των μοναχικών
και θα καταστρέψει τους ιερούς ναούς των αμαρτωλών. (ΙV-24)

Οι γυναίκες θα πάρουν, για άλλη μια φορά στην ιστορία της ανθρωπότητας, την εξουσία. Γενικότερα, οι θηλυκές δυνάμεις του σύμπαντος θα αποδεσμευθούν και θα εκφραστούν. Αλλά και η μεγάλη μητέρα, η μητέρα Γη, θα πάρει τη θέση της στις προτεραιότητες των ανθρώπων.


Μετά θα έρθει από μια μακρινή χώρα
ένας Γερμανός πρίγκιπας πάνω σε χρυσό θρόνο.
Η δουλεία και η θάλασσα θα συναντηθούν.
Η κυρία υποταγμένη, δεν την λατρεύουν πια. (II-87).


Η «κυρία που δεν λατρεύουν πια» είναι η θηλυκή όψη της θεότητας η οποία έχει για καιρό παραμεληθεί ή αγνοηθεί. Όταν αποκατασταθεί η ειρήνη, θα αποκατασταθεί και η αδικία μιας μακροχρόνιας πατριαρχικής εποχής να υποτιμήσει ή να αγνοήσει τη λατρεία της θηλυκής θεότητας. Αυτό που θα κατανοήσουν οι άνθρωποι είναι ότι το άφυλο της θεότητας προέρχεται από τη σύνθεση των δύο φύλων΄ έτσι θα αποκτήσουν μία πιο ισορροπημένη άποψη για το σύμπαν και τις δυνάμεις του.
