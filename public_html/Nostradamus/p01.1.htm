1. Εισαγωγικά

1.1 Ο Νοστράδαμος ερμηνεύει τον εαυτό του                                                                          

«Ως άνθρωπος που επιφορτώθηκα το βάρος του μέλλοντος, είμαι έγκλειστος μέσα στις μελλοντικές αναμνήσεις μου. Είναι μια πραγματική φυλακή καθώς δεν μπορώ να απελευθερώσω το πνεύμα μου απ' όλα όσα είδα. Παραμένω και θα παραμένω δεμένος με τις σκέψεις μου. Δεν «βλέπω» απλώς τα γεγονότα. Έχω μία βιωματική εμπειρία με το κάθε γεγονός, η οποία συνοδεύεται από έντονα συναισθήματα. Δεν είναι σαν να παρακολουθώ μία ταινία στο σινεμά΄ επηρεάζομαι βαθύτατα και θα ήταν απίθανο να μη μου συμβαίνει κάτι τέτοιο. Όσες φορές προσπάθησα να σταματήσω να βλέπω το μέλλον, με κατακυρίευε η περιέργεια για το τι θα συμβεί, κι έτσι έζησα τη ζωή μου μέσα σ΄αυτή την αμφιθυμία.
Όλη μου η ζωή διεξάγεται μέσα στο μυαλό μου. Τα σπουδαιότερα γεγονότα της ζωής μου διαδραματίζονται μέσα στο κεφάλι μου΄ όμως αυτή δεν είναι η πραγματική ζωή;»

Όπως ακριβώς ο ίδιος ο Νοστράδαμος είχε προφητέψει, πριν από πέντε αιώνες, ότι δηλαδή θα επανεμφανιζόταν ασώματος μία και μόνη φορά στο μέλλον του ιστορικού χρόνου, για να ερμηνεύσει ο ίδιος τις προφητείες του, έτσι και έκανε. Με τη μέθοδο της ύπνωσης , με την οποία είναι δυνατόν να έρθει κανείς σε επαφή με όντα άλλων διαστάσεων και την επιμελέστατη εργασία της Ντολόρες Κάννον, ο Νοστράδαμος εμφανίστηκε και μίλησε μέσα από τη φωνή ενός αγωγού. Η εργασία διήρκεσε δέκα χρόνια, από το 1982 μέχρι το 1992, στη διάρκεια των οποίων πέρασαν από τη θέση του αγωγού πολλοί άνθρωποι πρόθυμοι να βοηθήσουν το έργο αυτό΄ άνθρωποι με διαφορετικό μορφωτικό υπόβαθρο, ψυχοσύνθεση και προσανατολισμό ζωής.


Όπως εξομολογήθηκε ο ίδιος, αισθανόταν ότι ήταν πολύ σημαντικό να ανοίξει μία γραμμή επικοινωνίας με τη γη, διότι αυτή την εποχή τα πράγματα είναι ιδιαίτερα κρίσιμα. Βρίσκεστε σ' ένα σημείο, είπε, που αυτά τα γεγονότα θα συμβούν στη διάρκεια της παρούσας ενσάρκωσής σας και θα έχουν βαθιές επιπτώσεις στη ζωή όλων των ανθρώπων. Θα είναι μία πολύ τραυματική εποχή. Οι ψυχές που βρίσκονται στη γη αυτό τον καιρό ήταν ενήμερες για τις συνέπειες πριν ενσαρκωθούν σ' αυτή τη ζωή.

Οι άνθρωποι θα χρειάζονται σταθερότητα και πίστη σ' ένα σκοπό για να καταφέρουν να τα βγάλουν πέρα αυτές τις στιγμές. Επιλέξατε να βρίσκεστε εδώ, την εποχή αυτή. Η αναλογία παλαιών ψυχών στον κόσμο είναι τώρα μεγαλύτερη από οποτεδήποτε άλλοτε, επειδή οι παλιές ψυχές θα αποδειχτούν χρήσιμες για την επιβίωση του κόσμου. Θα τις βρείτε παντού, στα πιο απίθανα μέρη. Οι παλιές ψυχές θα βρίσκονται σε επικοινωνία μεταξύ τους και θα είναι αυτές που θα βοηθήσουν την κατάσταση να κρατηθεί και να επιβιώσει.

Η αχτίδα ελπίδας είναι υπαρκτή και ο σκοπός του όταν επέστρεφε για να ερμηνεύσει τα τετράστιχα ήταν να προσπαθήσει τουλάχιστον να τροποποιήσει, αν όχι να αποτρέψει τις χειρότερες εκφάνσεις των γεγονότων που πρόκειται να συμβούν. Είτε τροποποιηθούν τα γεγονότα είχε όχι, ακόμα κι αν πρόκειται να συμβούν στη χειρότερη εκδοχή τους, θα υπάρξει μία τεράστια πνευματική αναγέννηση σ' ολόκληρο τον κόσμο.

Και μάλιστα, κατά την περίοδο των ταραχών, οι άνθρωποι, εξατομικευμένα, θα έχουν ευκαιρίες για να έρθουν σ' επαφή με τον εαυτό τους και να συνειδητοποιήσουν ότι οι υλικές αξίες ήταν ψεύτικες. Μετά την περίοδο των ταραχών, όταν οι άνθρωποι θα αρχίσουν να επικοινωνούν ξανά μεταξύ τους, θα ανακαλύψουν κι άλλους συντρόφους στον πνευματικό δρόμο. Το γεγονός αυτό θα προκαλέσει μία τεράστια αναγέννηση της φιλοσοφίας και μία μεγάλη μίξη των καλύτερων στοιχείων των ανατολικών και δυτικών θρησκειών, θα καταλήξει σ' ένα παγκόσμιο κίνημα φιλοσοφικής σκέψης, το οποίο θα είναι εναρμονισμένο μ' αυτά που οι άνθρωποι γνωρίζουν και αισθάνονται ως αληθινά. Έτσι θα έρθουν στην επιφάνεια οι καλύτερες όψεις της Εποχής του Υδροχόου. Αν οι άνθρωποι κατάφερναν να το συνειδητοποιήσουν αυτό έγκαιρα και να κρατηθούν απ' αυτή την αχτίδα ελπίδας, θα μπορούσαν τότε ν' αποφύγουν μερικές από τις χειρότερες πλευρές της περιόδου αυτής των ταραχών που έπεται. Αυτό, όμως, είναι απίθανο να συμβεί σε ευρεία κλίμακα, εξαιτίας των υλιστικών αξιών, τις οποίες έχει η πλειονότητα των ανθρώπων στη γη.
Τέλος, ο Νοστράδαμος ελπίζει πως αυτά τα μηνύματα, που θα προσπαθήσει να μεταδώσει, θα φτάσουν τη σωστή στιγμή εκεί που πρέπει. Ελπίζει οι άνθρωποι να είναι αρκετά ανοιχτοί για να τα δεχτούν και ίσως να τα διαδώσουν ώστε να βοηθήσουν να αναστραφούν οι καταστροφές που έχει δει, γιατί είναι αναστρέψιμες.
