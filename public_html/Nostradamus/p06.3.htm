6.3 Ο εκφυλισμός ηγετών και νόμων

Η ψεύτικη τρομπέτα που κρύβει τρέλα
θα αναγκάσει το Βυζάντιο να αλλάξει τους νόμους του.
Από την Αίγυπτο θα πορευτεί ένας άντρας που
θέλει να αποσυρθεί το διάταγμα, αλλάζοντας χρήματα και σταθμά. (I-40)


«Η ψεύτικη τρομπέτα» αναφέρεται σε ισχυρούς ηγέτες, πολιτικούς όσο και θρησκευτικούς. Άνθρωποι που έβγαλαν το ψωμί τους αναμειγνυόμενοι με τη θρησκεία, ενώ προηγουμένως στη ζωή τους ήταν αναμειγμένοι με την πολιτική. Θρησκευτικοί άνδρες δογματικού χαρακτήρα, οι οποίοι διαστρεβλώνουν τον λόγο του Θεού και τον χρησιμοποιούν για δικούς τους σκοπούς. Αρκετοί απ' αυτούς αγωνίζονται για πολιτική εξουσία και συσπειρώνονται γερά προκειμένου να βοηθήσουν όσο περισσότερους του είδους τους μπορούν, να καταλάβουν θέσεις-κλειδιά στην κυβέρνηση.

Πολλές απ' τις θέσεις αυτές δεν είναι απαραίτητα θέσεις γοήτρου ή δημοσιότητας. Μπορεί κάλλιστα να είναι μια ήσυχη θεσούλα, χαμένη στα γρανάζια της γραφειοκρατίας σε κάποιο, όμως, νευραλγικό σημείο όσον αφορά την πληροφόρηση και την εξουσία, απ' όπου μπορούν να εργάζονται για τους σκοπούς τους και να επηρεάζουν αδιόρατα τα παγκόσμια γεγονότα προς όφελός τους. Αυτοί οι άνθρωποι, καθώς θα αποκτούν πολιτική δύναμη, θα επηρεάσουν όλο τον κόσμο. Θα προκληθούν επιπτώσεις σε πολλές χώρες στη Μέση Γη, εννοώ την ανατολική Ευρώπη, τη δυτική Ασία και τη Μέση Ανατολή.

Οι ηγέτες, σ' αυτό το τμήμα του κόσμου, θα θορυβηθούν ιδιαίτερα από την τροπή που θα πάρουν τα πράγματα. Θα αρχίσουν να αλλάζουν τους νόμους τους ως αντίδραση στην κατάσταση, καθιστώντας πιο δύσκολο για τους Αμερικανούς να ταξιδέψουν σ' αυτή την περιοχή του κόσμου - «το Βυζάντιο θα αναγκαστεί να αλλάξει τους νόμους του». Μερικοί νόμοι ειδικότερα, που θα αλλάξουν, θα σχετίζονται με την ανταλλαγή του Αμερικανικού νομίσματος με άλλα νομίσματα και στο εμπόριο με τις Ηνωμένες Πολιτείες. Θα υπάρξουν αρνητικές συνέπειες. Όλα αυτά θα έχουν ως αποτέλεσμα να επηρεαστεί ο νεαρός Αντίχριστος, ο οποίος αναφέρεται ως «Βυζάντιο». Ο νεαρός Αντίχριστος, στη χώρα του, όντας στη διαδικασία να οικοδομήσει μία βάση εξουσίας, θα επηρεαστεί από τις διεστραμμένες ενέργειες αυτών των δογματικών με τέτοιο τρόπο που αυτό, αργότερα, θα δυσκολέψει τα πράγματα για τη Χριστιανοσύνη γενικότερα.

Οι δογματικοί, θα γίνουν, έτσι, οι ίδιοι παράγοντες της δικής τους καταστροφής. Αυτοί οι άνθρωποι που εμφανίζονται πολύ θρήσκοι είναι πολύ πονηροί και υπολογιστές. Όταν παρακολουθούν εκπαιδευτικά σεμινάρια, για παράδειγμα πώς να γίνουν δεσπότες και τα παρόμοια, πολλά απ' αυτά που μαθαίνουν μπορούν να χρησιμοποιηθούν για να ελέγξουν τις μάζες, να κάνουν πλύση εγκεφάλου και να χειραγωγούν τους ανθρώπους. Αυτό είναι βασικά αυτό που κάνουν, κι αυτά είναι προσωπικά, κοσμικά πράγματα αντί να είναι μόνο θρησκευτικά. Αυτό που στην πραγματικότητα επιδιώκουν, είναι η εξουσία.
