9.10 Ανακούφιση από τους διαστημικούς σταθμούς


Θα είναι το τέλος του πολιτισμού όπως τον ξέρατε μέχρι τώρα, αλλά δεν θα είναι το τέλος της ανθρωπότητας. Δραστικές αλλαγές θα συμβούν, αλλά η ανθρωπότητα είναι αρκετά προσαρμοστική ώστε να υιοθετήσει έναν νέο τρόπο ζωής και να συνεχίσει. Δεν θα επιτρέψει να εξαφανιστεί η ζωή από τον πλανήτη αν είναι στο χέρι της να κάνει κάτι γι' αυτό.

Όταν το ψάρι που ταξιδεύει σε στεριά και θάλασσα
θα βγει από ένα μεγάλο κύμα στη στεριά,
το σχήμα του θα είναι ξένο, λείο και τρομακτικό.
Από τη θάλασσα οι εχθροί σύντομα φτάνουν στα τείχη. (Ι-29).

«Το ψάρι που ταξιδεύει σε στεριά και θάλασσα» είναι αυτό που οι άνθρωποι έχουμε ονομάσει Ούφο. Τη συγκεκριμένη εποχή θα υπάρξει στενότερη επαφή με τις δυνάμεις πίσω από αυτά τα σκάφη. Ένα από αυτά, καθώς θα κατευθύνεται προς μία  υποθαλάσσια βάση, θα παρουσιάσει βλάβη και «θα βγει στη στεριά». Πολλοί άνθρωποι θα φοβούνται τα αντικείμενα αυτά΄τα θεωρούν «εχθρούς που σύντομα φτάνουν στα τείχη».


Για να ανακουφιστούν από τις επιδράσεις αυτών των καταστροφών οι άνθρωποι θα πρέπει να συνεχίσουν να εξελίσσουν καινούργια τεχνολογία και διαστημικά προγράμματα πριν συμβεί η μετατόπιση του άξονα της γης. Από τις διάφορες εναλλακτικές λύσεις που μπορεί να δει, αυτή που θα υποστεί τις λιγότερες επιδράσεις είναι οι άνθρωποι να έχουν αναπτύξει διαστημικούς σταθμούς, οι οποίοι θα κινούνται γύρω από τη γη πριν να γίνει η μετατόπιση.

Γιατί ένα από τα βασικά προβλήματα θα είναι να βρεθεί ηλεκτρικό για να λειτουργήσουν τα πράγματα, ηλιακοί σταθμοί που θα μπορέσουν να παράγουν ενέργεια για να επιταχυνθούν οι διαδικασίες ανοικοδόμησης. Αν η τεχνολογία στην επιφάνεια της γης είναι περιορισμένη, η γη θα καταστραφεί σχεδόν τελειωτικά. Αλλά αν οι κυβερνήσεις προνοήσουν επιτρέποντας στον άνθρωπο να μετακινηθεί στο σύμπαν, τότε υπάρχουν ακόμη ελπίδες να μη χαθούν τα πάντα. Υπάρχει η πιθανότητα οι «Άλλοι» να μας βοηθήσουν κατά την διάρκεια της μετακίνησης, αλλά αυτό εξαρτάται από το πώς θα χειριστούμε εμείς την κατάσταση. Αν προσπαθήσουμε να εργαστούμε ενωμένοι με όσο τον δυνατόν πιο θετικό τρόπο, θα έρθουν να μας βοηθήσουν. Αν αντιδράσουμε αρνητικά, εκτοξεύοντας πυρηνική ενέργεια και άλλα τέτοια, επειδή θεωρούμε πως ήρθε το τέλος του κόσμου, τότε θα μείνουν μακριά.

Όσοι ζουν σήμερα σε ορεινή περιοχή θα πρέπει να επιβιώσουν από τους σεισμούς που θα γίνουν εκεί. Οι άνθρωποι στις πεδιάδες θα είναι περισσότερο ασφαλείς από τους σεισμούς, αλλά όταν αρχίσουν να λιώνουν οι πάγοι, οι άνθρωποι στα βουνά θα είναι πιο ασφαλείς. Αυτή θα είναι μια αμφίρροπη διαδικασία. Με τον καιρό οι άνθρωποι θα γίνουν πιο ευαίσθητοι σε εσωτερικές προειδοποιήσεις και δεν θα αιφνιδιάζονται τόσο. Θα είναι πιο προετοιμασμένοι γιατί θα μάθουν να ακούν και να εμπιστεύονται τις προειδοποιήσεις του εσωτερικού τους εαυτού.
Προσέξτε τη σειρά των γεγονότων: οι σεισμοί έρχονται πάντα πρώτοι΄ σεισμοί και εκρήξεις ηφαιστείων γίνονται ταυτόχρονα. Πιθανόν να γίνουν τόσες πολλές καταστροφές στην Ευρώπη και στην Ασία, όσες και στις Ηνωμένες Πολιτείες όσον αφορά στους σεισμούς. Παρόλ' αυτά, οι χώρες αυτές δεν θα βυθιστούν, δεν θα πλημμυρίσουν με τόσο πολύ νερό όσο οι Ηνωμένες Πολιτείες. Οι χώρες αυτές θα βρίσκονται κοντά και θα είναι πιο εύκολο να ανοικοδομηθούν.

Τα κεντρικά της εξερεύνησης του διαστήματος θα φύγουν από την Ουάσιγκτον-Όρεγκον περιοχή. Η ΝΑΣΑ και τα πάντα θα εξαφανιστούν από την Φλόριντα. Οι εθνικές κυβερνήσεις δεν θα είναι απασχολημένες με το να διοικούν τους ανθρώπους, όπως κάνουν τώρα, αλλά θα είναι πιο συνδεδεμένες με διαστημικά εγχειρήματα. Αυτός θα είναι πιο πολύ ο σκοπός των κυβερνήσεων, παρά να επιβάλλουν νόμους για ιδιώτες όπως κάνουν τώρα. Θα εστιάσουν την προσοχή τους κατά κύριο λόγο στο διάστημα. 
Τα διαπλανητικά ταξίδια, και η μεταφορά ανθρώπων από άλλους πλανήτες θα είναι πιο σημαντικά.
Εκλογές δεν θα γίνονται κατά τον ίδιο τρόπο όπου διάφορα κόμματα αντιμάχονται. Θα υπάρχει κάτι σαν συμβούλιο που θα έχει την υποστήριξη όλων των ανθρώπων και όλοι θα εργάζονται για το κοινό καλό. Οι κυβερνήσεις δεν θα εστιάζουν στον έλεγχο των ανθρώπων, αλλά στην άνοδο, γιατί ελέγχουν οι ίδιοι οι άνθρωποι τους εαυτούς τους, ανεξάρτητα ή εξατομικευμένα. Δεν θα χρειάζονται πια οι κυβερνήσεις να το κάνουν αυτό για τους ανθρώπους.

Όλες αυτές οι δραστικές αλλαγές δεν θα προκληθούν μόνο από τη μεγάλη μετατόπιση του άξονα της γης. Θα συνέβαιναν ανεξάρτητα, εξαιτίας της εξέλιξης της ανθρωπότητας. Η ανθρωπότητα θα στραφεί όλο και πιο πολύ στη διαίσθηση και στο πνεύμα, κάτι που θα γίνει την ίδια περίπου περίοδο με την αλλαγή της θέσης του άξονα. Στην ουσία θα συνεργάζονται. Και όταν λέω την ίδια περίπου περίοδο, μιλώ για δύο ή τρεις δεκαετίες. Η συνειδητοποίηση της διαίσθησης θα εξελίσσεται όλο και πιο πολύ μέχρι που να γίνει αποδεκτή. Αυτή τη στιγμή εσύ ασχολείσαι με αυτήν, αλλά δεν έχει αγγίξει σε ικανοποιητικό βαθμό τους ανθρώπους.

Αυτά που σας λέω τώρα, θα έχουν γίνει αποδεκτά μέχρι τότε σε σημείο που να έχετε συνειδητοποιήσει και να γνωρίζετε για τη μετατόπιση του άξονα. Θα την αναμένετε και θα έχετε προετοιμαστεί για να ανταπεξέλθετε.
Μιλάμε για έναν κόσμο που το χάος έχει αλλάξει τα πάντα, για πάντα΄ έναν πλανήτη διαμορφωμένο πιο πολύ από νερό παρά ξηρά. Μια εικόνα του κόσμου που ελπίζω πως δεν θα την δουν ποτέ οι άνθρωποι.
