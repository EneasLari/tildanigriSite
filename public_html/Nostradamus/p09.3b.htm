9.3β Η Αγγλία    


Μόνο τα βουνά μεταξύ Αγγλίας και Σκωτίας θα παραμείνουν. Και αυτά τα βουνά της Σκωτίας θα είναι απλά σκόρπια νησάκια. Μπορεί να υπάρχουν μερικά μικρά νησάκια εκεί που βρισκόταν η Ιρλανδία, αλλά θα είναι πολύ μικρά.

Η Μεγάλη Βρετανία, που θα περιλαμβάνει την Αγγλία,
θα καλυφθεί από τεράστιες πλημμύρες.
Η νέα ένωση στην Ausonne θα κάνει πόλεμο
έτσι ώστε θα συμμαχήσουν εναντίον τους. (ΙΙΙ-70)

Λόγω της μετατόπισης του άξονα, θα πλημμυρίσει όλη η χώρα, οι κάτοικοι θα μετακομίσουν στα ορεινά και στα εδάφη γύρω από τις Γαλλικές ¶λπεις, αναζητώντας εδάφη για να κατοικήσουν. Θα υπάρξουν διαφορές στη νοοτροπία και τις απόψεις, μέχρι οι άνθρωποι να προσαρμοστούν στη νέα πραγματικότητα.

Οι μακρυμάλληδες πολίτες της Κελτικής
μαζί με ξένα έθνη
θα αιχμαλωτίσουν το λαό της Ακουϊτανίας
έτσι ώστε να υποκύψουν στα σχέδιά τους. (ΙΙΙ-83)

Η Αγγλία μετά την καταστροφή έχει την εικόνα ενός πολύ μικρού νησιού. Η Αγγλία θα ήθελε να έχει εδάφη προς την πλευρά της Γαλλίας. Ανάμεσα στις δύο χώρες έχει δημιουργηθεί κάτι σαν φέουδο. Αυτό έχει συμβεί και παλαιότερα στην ιστορία. Η Ακουϊτανία υπήρξε περιοχή της νοτιοδυτικής Γαλλίας.
