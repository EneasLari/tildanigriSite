10.8 Τα βαρυτικά πηγάδια

Οι μετανάστες και οι εξόριστοι ανακαλούνται πίσω,
οι πατέρες και οι γιοί δυναμώνουν τα βαθιά πηγάδια.
Ο άσπλαχνος πατέρας και οι ακόλουθοί του ασφυκτιούν.
Ο πιο μοχθηρός γιος του πνίγηκε στο πηγάδι. (IV-53)

Στο μακρινό μέλλον, οι διαστημικές αποικίες και οι διαστημικές πτήσεις θα είναι κάτι συνηθισμένο, επειδή η γη θα είναι πυκνοκατοικημένη. Κάποια στιγμή, οι κάτοικοι της γης θα αντιμετωπίσουν προβλήματα και θα καταφύγουν στις αποικίες για βοήθεια. Οι αποικίες θα μπορούν τότε να βοηθήσουν τους γήινους, με υπηρεσίες και αντικείμενα που κατασκευάζονται αποκλειστικά στο διάστημα με τεχνογνωσία που δεν θα υπάρχει πλέον στη γη.


Η φράση «δυναμώνουν τα βαθιά πηγάδια» αναφέρεται στα βαρυτικά πηγάδια, που βρίσκονται γύρω από κάθε πλανήτη. Αυτό σημαίνει πως πρέπει να ξεφύγεις από την επιρροή της βαρύτητας για να φύγεις από τον πλανήτη και πρέπει να την υπολογίσεις όποτε κινείσαι γύρω του. Για να ενδυναμώσουν αυτά τα πηγάδια θα στείλουν υλικά και αποθέματα στις δυνάμεις της γης που τα έχουν ανάγκη.

Κατά τη διαδικασία αυτής της ανταλλαγής ένας άνδρας που εισέρχεται ξανά στην ατμόσφαιρα κάνει έναν λάθος υπολογισμό και καίγεται -«ο γιος του πνίγηκε στο πηγάδι». «Ο άσπλαχνος πατέρας και οι ακόλουθοί του ασφυκτιούν» περιγράφει αυτούς που προσπάθησαν να καταστρέψουν τις ζωές πολλών ανθρώπων και να επαναφέρουν την εποχή των ταραχών αλλά δεν θα τα καταφέρουν. Οι άνθρωποι των αποικιών, που βρίσκονται ήδη στο διάστημα, θα είναι αυτοί που τελικά θα συλλάβουν τον «άσπλαχνο πατέρα» καθώς αυτός θα προσπαθεί να διαφύγει από τη γη με ένα διαστημόπλοιο.
