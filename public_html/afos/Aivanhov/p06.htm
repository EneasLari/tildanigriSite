«Η δύναμη της σκέψης»

Οι σκέψεις είναι ζωντανές και δρώσες οντότητες

Οι άνθρωποι δεν είναι συνηθισμένοι να αντιμετωπίζουν τις σκέψεις σαν ζωντανές οντότητες. Αυτό δεν είναι γραμμένο πουθενά στην επίσημη επιστήμη. Πρόκειται για ένα τομέα εντελώς άγνωστο. Μόνον η Επιστήμη των Μυστών, που έχει μελετήσει αυτό το θέμα πολύ καλά, διδάσκει ότι οι σκέψεις είναι ζωντανές οντότητες εξηγώντας μάλιστα ότι εκείνος που τις δημιουργεί δεν είναι ο άνθρωπος. Οι σκέψεις μπορούν να έρθουν να εγκατασταθούν μέσα του για να τον βοηθήσουν ή να τον καταστρέψουν, αλλά δεν τις δημιουργεί ο ίδιος.

Εκείνος δημιουργεί μόνο τις δυνατότητες για να έρθουν να τον επισκεφτούν. Με τις σκέψεις συμβαίνει  ό,τι ακριβώς συμβαίνει με τα παιδιά. Ο άντρας και η γυναίκα δεν μπορούν ποτέ να δημιουργήσουν ένα παιδί, δηλαδή το πνεύμα του. Αυτοί χτίζουν μόνο την κατοικία του, το φυσικό του σώμα, όπου το πνεύμα θα έρθει να κατοικήσει. Αυτή η κατοικία μπορεί να είναι ένα καλύβι, ένα παλάτι ή ένας ναός, ανάλογα με τα υλικά που κατάφεραν να προμηθευτούν.

Έτσι λοιπόν, ο άνθρωπος δεν δημιουργεί τις σκέψεις, μόνο τις προσελκύει ή τις απωθεί, εφόσον και σ΄αυτόν τον τομέα ισχύουν οι νόμοι της έλξης και της άπωσης. Αν μπορούσε ο ίδιος να τις δημιουργήσει θα έπρεπε να μπορεί και να τις καταστρέψει, αλλά δεν είναι έτσι, αυτό το γνωρίζετε καλά και από προσωπική εμπειρία. Πόσες φορές δε ρίχνονται πάνω σας σκέψεις σαν σφήκες ή σαν μύγες και είναι αδύνατο να τις ξεφορτωθείτε! Γιατί; Επειδή εσείς δημιουργήσατε τις συνθήκες για να τις έλξετε. Αφήσατε να περιφέρονται μέσα σας ακαθαρσίες κι αυτό προσελκύει πλάσματα που αυτές οι βρωμιές τούς αρέσουν. Εξαγνιστείτε και τότε θα δείτε τι σκέψεις θα έρθουν! Υπάρχουν σκέψεις σε όλες τις περιοχές του διαστήματος... μέχρι τον κόσμο των Ιδεών για τον οποίο μιλάει ο Πλάτωνας. Τι είναι οι ιδέες που αναφέρει ο Πλάτων; Είναι αιώνιες Αρχές, Αρχέτυπα, Δυνάμεις οι οποίες εργάζονται για να διαμορφώσουν και να διαπλάσουν το σύμπαν. Είναι θεότητες. Κάθε Ιδέα είναι μία θεότητα.
