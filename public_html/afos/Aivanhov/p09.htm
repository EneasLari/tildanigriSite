«Η δύναμη της σκέψης»

Η φωνή μας εισακούγεται στα πέρατα του κόσμου

Γιατί πρέπει να το ξέρετε: όταν ένας άνθρωπος έχει αφιερώσει αληθινά τη ζωή του στο φως, η εργασία του έχει αποφασιστική σημασία για τις υποθέσεις του κόσμου. Όπου και να βρίσκεται, γνωστός ή άγνωστος, είναι ένα κέντρο, μία εστία τόσο ισχυρή που τίποτα δεν γίνεται χωρίς αυτόν. Εναρμονίζει τις δυνάμεις του σύμπαντος σ' ένα σκοπό φωτεινό, κι ακόμα συμμετέχει και στις αποφάσεις των πνευμάτων εκεί ψηλά. Σας εκπλήσσει αυτό; Κι όμως είναι φυσικό.

Γιατί τα φωτεινά πνεύματα που επιτηρούν το πεπρωμένο του κόσμου δεν θα έπαιρναν υπόψη τους τη γνώμη άλλων πνευμάτων που τους μοιάζουν με τη λάμψη και τις εκπορεύσεις τους;
Αν, όταν πρόκειται να ληφθούν αποφάσεις για το μέλλον της ανθρωπότητας, δεν μπορούσε κανένας εδώ στη γη να εκφράσει τη γνώμη του, αυτό δε θα ήταν ούτε λογικό ούτε σωστό. Πρέπει λοιπόν από ΄δω και πέρα να μάθετε ότι η φωνή σας μπορεί να ακουστεί για μια απόφαση για το μέλλον του κόσμου, και πρέπει ακόμα να ξέρετε πως μπορείτε να συμμετέχετε στα συμβούλια εκεί πάνω. Από εκείνη τη στιγμή η ζωή σας θα πάρει καινούργιο νόημα. Θα καταλάβετε καλύτερα πόσο σημαντικό είναι να αρχίσετε να ζείτε μια θεϊκή ζωή που θα σας κάνει άξιους να έχετε φωνή που να ακούγεται στον κόσμο των πιο υψηλών οντοτήτων.

Θα πείτε: «Όμως ο μαθητής έχει συνείδηση του ρόλου του;» Μπορεί να την έχει, όμως σίγουρα όχι από την αρχή. Μέσα του υπάρχει κάτι που συμμετέχει, που το λαμβάνουν υπόψη, που το ακούν, αλλά αυτά συμβαίνουν στις ανώτερες σφαίρες της συνείδησής του στις οποίες η κοινή συνείδηση δεν μπορεί να εισχωρήσει. Το φυσικό επίπεδο είναι τόσο πολύ θαμπό, πυκνό, που χρειάζεται πολύς χρόνος και πολλή προσπάθεια ώστε αυτά που συμβαίνουν στις ουράνιες περιοχές να έρθουν να αντανακλαστούν σ' αυτό.
