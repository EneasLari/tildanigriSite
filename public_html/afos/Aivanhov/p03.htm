«Νίκη του πνεύματος: η ελευθερία»

Πρέπει να συνδεθείτε πριν αποσπαστείτε

Και πόσους ανθρώπους δεν είδα στη ζωή μου να προσπαθούν να ελευθερωθούν, αλλά με τέτοιο αδέξιο τρόπο, τόσο επικίνδυνο και γι΄ αυτούς και για τους άλλους! Τους έλεγα: «Κοιτάξτε τώρα: έχετε ένα παλιό σπίτι που δεν σας αρέσει, δεν είστε καλά μέσα σ' αυτό, σύμφωνοι, το καταλαβαίνω. Πριν όμως το γκρεμίσετε, πρέπει να χτίσετε ένα άλλο, διαφορετικά θα βρεθείτε στο δρόμο, μέσα στη βροχή, μιλώντας συμβολικά».

Είναι ακριβώς αυτό που σας εξηγώ όταν σας λέω ότι πρέπει να συνδεθείτε πριν αποσπαστείτε. Αυτή η σύνδεση είναι το χτίσιμο μιας νέας, πιο όμορφης κατοικίας. Στη συνέχεια, μπορείτε να αποσπαστείτε, δηλαδή να γκρεμίσετε το παλιό σπίτι. Δεν πρέπει όμως να κάνετε το αντίθετο, δηλαδή να αποσπαστείτε πριν να έχετε συνδεθεί, επειδή η απόσπαση δεν θα εξυπηρετήσει τίποτα, θα σας δέσει με όλων των ειδών τα ετερόκλητα πράγματα και θα είστε ακόμα πιο δυστυχείς. Δεν πρέπει να αποκόβεστε πριν να έχετε συνδεθεί, δεν πρέπει να καταστρέφετε πριν να έχετε χτίσει.

Για να σας αποδείξω ότι όταν σας λέω κάτι, μπορείτε να το επαληθεύσετε σε όλες τις εκδηλώσεις της φύσης, θα σας δώσω ακόμα ένα παράδειγμα. Χτυπήσατε κάπου και στην πληγή έχει σχηματιστεί ένα ξερό τμήμα δέρματος. Από κάτω αρχίζει να σχηματίζεται ένα νέο δέρμα, αν όμως βγάλετε το ξεραμένο τμήμα πριν το νέο δέρμα να έχει σχηματιστεί εντελώς, η πληγή σας θα ξανανοίξει και θα πρέπει και πάλι να περιμένετε την ίδια διαδικασία. Πριν να βγάλετε το ξερό δέρμα, πρέπει οπωσδήποτε «να χτίσετε το σπίτι», δηλαδή να αφήσετε το νέο δέρμα να σχηματιστεί από κάτω.

Αν ο άνθρωπος ήξερε να παρατηρεί, θα έβλεπε ότι ο οργανισμός, η φύση, τα δέντρα, ολόκληρο το σύμπαν, εργάζονται με τον ίδιο τρόπο, εκτός από τους ανθρώπους.
Οι άνθρωποι θέλουν να είναι ελεύθεροι, καθώς όμως δεν γνωρίζουν τον κίνδυνο του να αποσπαστούν πριν να έχουν συνδεθεί, πάντοτε τους συμβαίνει κάτι που δεν το πρόβλεψαν. Ναι, αν δεν βάλετε μέσα στην καρδιά, μέσα στο μυαλό σας, την ύπαρξη ανώτερων πραγματικοτήτων, υπάρχουν άλλες οι οποίες θα έρθουν να διεισδύσουν μέσα σας για να πάρουν τη θέση, κι αυτό θα είναι φοβερό. Πρέπει λοιπόν η καρδιά και το μυαλό σας να είναι απασχολημένα από ένα υπέροχο ιδανικό, απ' ό,τι καλύτερο, ό,τι πιο ευγενικό, ό,τι πιο φωτεινό υπάρχει. Γι' αυτό οι αληθινοί Μύστες διδάσκουν στους μαθητές τους να βάζουν το Θεό πάντοτε στην πρώτη θέση. Ακόμα κι αν αυτό σας φανεί γελοίο, ανόητο, αντίθετο με όλες τις σύγχρονες και μοντέρνες ιδέες, κάντε το, μάθετε να βάζετε μέσα σας το Θεό στην πρώτη θέση.

Οι περισσότεροι παιδαγωγοί αγνοούν τη σημασία αυτού του νόμου: να συνδέεται κάποιος πριν αποσπαστεί. Πρέπει να δείχνετε άλλα πράγματα, να προσφέρετε στους ανθρώπους να γευτούν νέα πράγματα, αυτό είναι που κάνω εγώ.
Ξέρω πολύ καλά ότι δεν θα καταφέρω να σας αποσπάσω από ορισμένες συνήθειες ή σκέψεις αν δεν κατορθώσω να σας κάνω να προσκολληθείτε σε κάτι άλλο. Γι' αυτό πρώτα σας παρουσιάζω όλα τα μεγαλείο του Ουρανού, της ψυχής, τους πνεύματος, του ήλιου, ώστε στο τέλος να πείτε: «Α! τι όμορφα, τι υπέροχα που είναι! Πόσο θάθελα να μείνω εδώ!» Κι όταν θα ξαναδείτε τα υπόλοιπα, τι φρίκη, θα το βάλετε στα πόδια! Πρέπει πάντοτε να οδηγούμε τους ανθρώπους να συνειδητοποιούν μόνοι τους τι είναι το καλύτερο.
