«Η σεξουαλική δύναμη ή ο φτερωτός Δράκοντας»

Ο χείμαρρος που παρασύρει όλα τα φράγματα

Καταλάβετε ότι το καλύτερο φάρμακο στα προβλήματα της αγάπης, είναι ακριβώς η αγάπη. Πολλοί ήρθαν κοντά μου και μου παραπονέθηκαν για ορισμένες αδιαθεσίες, για ορισμένες ανισορροπίες και τους είπα: «Γιατί σταματήσατε να αγαπάτε; Αυτές οι ταραχές έρχονται επειδή καταπνίξατε την αγάπη μέσα σας, τότε συμπιέστηκε και τα αναποδογύρισε όλα. Η αγάπη είναι ένας τρομερός χείμαρρος, όμως δεν το ξέρετε, δεν είστε αρκετά φωτισμένος, και παρασύρει όλα τα φράγματα.

Αν θέλετε να σωθείτε από αυτά τα βάσανα, πρέπει να αγαπάτε, να αγαπάτε νύχτα-μέρα, να αγαπάτε όλα τα όντα ... δεν θα μείνει μάλιστα πια καιρός στη συνέχεια για να βασανίζεστε, θα είστε τόσο απασχολημένος!

Ενώ τώρα, όσο κλείνεστε στον εαυτό σας, τόσο τσιγγουνεύεστε την αγάπη, και τόσο τα πράγματα γίνονται πιο δύσκολα. Να είστε γενναιόδωροι, Θεέ μου, και θα σωθείτε, δίνετε την αγάπη σας σε όλα τα όντα. Εγώ αυτό κάνω, βρήκα το μυστικό. Βέβαια περνιέμαι λίγο βλάκας και όλος ο κόσμος λέει: «Ω! ο καημένος, με την καρδιά που έχει...»  Μα ακριβώς, με την καρδιά μου βρήκα το μυστικό που εσείς δε βρήκατε ακόμη με την τρομερά αναπτυγμένη νόησή σας».
Πρέπει να θαυμάζετε και να ευχαριστείτε κάθε μέρα τον Ουρανό που υπάρχουν στη γη εκατομμύρια όμορφες γυναίκες που δεν έχετε ακόμη την ευτυχία να γνωρίσετε και να θαυμάσετε, και εκατομμύρια έξυπνοι και δυνατοί άντρες που δεν έχετε ακόμη βρει την ευκαιρία να συναντήσετε!

Πρέπει να το σκέφτεστε αυτό και να χαίρεστε ... Να χαίρεστε μόνο και μόνο που υπάρχουν και με την ιδέα ότι, μια μέρα, θα μπορείτε να τους δείτε, να τους μιλήσετε, να τους θαυμάσετε. Σας φαίνεται παράξενο, γιατί δεν είστε συνηθισμένοι να χαίρεστε για τέτοιες ιδέες;... Είναι αλήθεια, αυτός είναι ένας τρόπος να σκέφτεται κανείς πολύ ασυνήθιστος, αλλά τόσο αποτελεσματικός. Δεχθείτε τον και θα δείτε τα αποτελέσματα.

Θα πείτε: «Ναι, μα αυτά που μας λέτε εδώ πάνε ενάντια σε όλους τους κανόνες της ηθικής. Αν όλοι οι άνθρωποι αρχίσουν να αγαπούν όλες τις γυναίκες και αντίθετα, δεν θα υπάρχει πια οικογένεια». Μα ναι, θα υπάρξει μια μεγάλη οικογένεια, κι αυτό είναι όλο. Μήπως αυτό είναι τόσο κακό;... Καταλάβετέ με καλά, όταν λέω ότι οι άντρες πρέπει να αγαπούν όλες τις γυναίκες και οι γυναίκες να αγαπούν όλους τους άντρες, αυτό δεν σημαίνει ότι πρέπει να πολλαπλασιάσουν τις εμπειρίες και ότι οι σύζυγοι πρέπει να είναι άπιστοι ο ένας στον άλλον.

Όχι, πρέπει να είστε πιστοί, πρέπει όμως να ξέρετε επίσης ότι ένας και μόνο άντρας, μία και μόνη γυναίκα δεν θα μπορεί ποτέ να σας τα δώσει όλα, και ότι εσείς οι ίδιοι δε θα μπορείτε ποτέ επίσης να τα δώσετε όλα στο σύζυγό σας ή στη γυναίκα σας. Γι' αυτό πρέπει να ζείτε μαζί, να εργάζεστε μαζί, αλλά να αγαπάτε όλο τον κόσμο, να χαμογελάτε σε όλο τον κόσμο, και να αφήνετε επίσης τον άλλο ελεύθερο να το κάνει. Ναι, να αγαπάτε ο ένας τον άλλο, να είστε μαζί, να μη χωρίζετε, αλλά να διευρύνετε την αντίληψή σας για την αγάπη. Έτσι οι σύζυγοι και οι γυναίκες θα μάθουν να ευχαριστιούνται με χαρές επίσης αιθέριες.
