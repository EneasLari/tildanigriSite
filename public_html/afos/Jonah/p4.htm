Επιστρέφουμε οι παλιοί Εσσαίοι

(...) Δεν είμαστε εδώ για να εκπροσωπήσουμε τη θρησκεία των ανθρώπων. Είμαστε εδώ για να εκπληρώσουμε μια προφητεία ότι θα επιστρέψουμε και θα φέρουμε ένα μήνυμα. Γιατί γνωρίζουμε καλά ότι όσοι έχουν μάτια να δουν θα δουν, όσοι έχουν αυτιά ν' ακούσουν θ' ακούσουν και όσοι δεν έχουν δε θα δουν, δε θ' ακούσουν. Είναι πάντοτε θέμα προσωπικής επιλογής.

Παρ' όλα αυτά, ήταν σκοπός μας να επιστρέψουμε από την τελευταία κιόλας φορά που ήμασταν πάλι εδώ, την εποχή που θα ορίζατε ως «δύο χιλιάδες χρόνια πριν», τον καιρό των Εσσαίων. Δώσαμε τότε την προφητεία ότι θα επιστρέψουμε πάλι για να βοηθήσουμε την ανθρωπότητα στην αλλαγή από τη μια εποχή σε μια άλλη εποχή βαθύτερης επίγνωσης, αληθινής πνευματικότητας - όπου η ανθρωπότητα θα πάψει να τοποθετεί την πνευματικότητα έξω από τον εαυτό της, αλλά θ' αρχίσει να τη διεκδικεί εσωτερικά, όπου καθένας θα διεκδικεί την πνευματικότητα μέσα στον ίδιο του τον εαυτό.
